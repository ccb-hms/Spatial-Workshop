# MD workshop

These tutorials were originally created for a French workshop for medical doctors learning maths applied to cancer. They had some math lectures and other practical sessions before this course.

## Content

This directory contains two notebooks:
1. Introduction to GeoPandas, with a special focus on `Point` and `Polygon` geometries
2. Introduction to `sopa` on synthetic data, with a focus on segmentation and its limitations

## Extra ressources

Other tutorials can be found on the [Sopa](https://gustaveroussy.github.io/sopa/) or [Novae](https://mics-lab.github.io/novae/) documentations.
